05

Decided rulings

do not re-litigate
  1. On-demand only, synchronous POST, no Queues. One R2 get + one AI call; the 30s concern applied to the rejected on-upload path.
  2. Fresh POST returns cache server-side — zero AI calls. force:true is the quiet escape. Stale = contentVersion OR promptVersion mismatch. A failed run is retryable and never overwrites a stored summary.
  3. Azure gpt-4.1-mini primary, Workers AI 70B fallback (keyless forks get Summarize free via the committed ai binding). Provider resolved per attempt; an Azure error is a retryable failure, never a silent downgrade. Whisper stays on Workers AI.
  4. Cold-race double-generation accepted for v1 (last-write-wins upsert, both write the same content) — lease machinery rejected at team scale. Pinned by C23.
  5. Ask AI deferred (round 5). Amendment at plan sign-off: generator ships as plain summarizeSite() — the round-5 answerFromSite shape was dropped by user ruling; do NOT reintroduce it.
  6. Rate limit = SUMMARY_LIMITER binding (upload.ts pre-write idiom), key = user id, generation attempts only — cache hits are free. Chosen over row-timestamp cooldown at plan sign-off.
  7. API worker only. The content worker stays thin; nothing heavy may enter the schema.ts import chain (content-worker bundle trap).

08

Strategy

  1. All feature cases are new-module-spec — the modules don’t exist yet; each case is written red first and drives its slice of the module into existence. No horizontal slicing: one case → minimal impl → green → next.
  2. Seam tests land with their seam: S-4 (FK pragma) lands inside S1 and must keep the whole existing suite green before C15’s cascade legs count.
  3. Characterization pins before the refactor they guard: C31 (feed columns + budgets) is written against current feeds BEFORE the hasSummary fragment lands; the existing sites-viewer indexPath pin guards S-1’s move.
  4. Economic + security invariants first within each step: C18 (zero-AI cache hit) and C24 (access matrix) are authored immediately after their tracer bullets — they are the two invariants the feature cannot ship without.
  5. End of each phase: thermo-nuclear gate → fix findings → full suite green → one phase commit. Never start phase N+1 on an uncommitted phase N.
